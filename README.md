# 📊 Customer Shopping Behavior Analysis

## Overview
This project analyzes customer shopping behavior using data analytics techniques to uncover purchasing patterns, customer preferences, and sales trends. The analysis includes data cleaning, exploratory data analysis (EDA), SQL-based analysis, and interactive dashboard creation.

The goal of this project is to transform raw data into meaningful insights that help businesses make data-driven decisions.

---

## Dataset
The dataset contains information about customer purchases and includes attributes such as:

- Customer demographics (Gender, Subscription Status)
- Product categories
- Shipping types
- Purchase behavior
- Transaction details

This dataset is used to analyze patterns in customer preferences and buying habits.

---

## Tools & Technologies
The following tools were used in this project:

- Python
- SQL (PostgreSQL / MySQL / SQL Server)
- Power BI
- Gamma (for presentation)

---

## Project Workflow

### 1. Data Loading
The dataset was imported and prepared for analysis using Python.

### 2. Data Cleaning
Data preprocessing was performed to:
- Handle missing values
- Remove inconsistencies
- Standardize column formats

### 3. Exploratory Data Analysis (EDA)
EDA was conducted using Python to understand data distributions, relationships, and patterns.

### 4. SQL Analysis
SQL queries were used to analyze important metrics such as:
- Customer purchasing patterns
- Category popularity
- Shipping method preferences
- Subscription impact on purchases

### 5. Dashboard Creation
Interactive dashboards were created using Power BI and Tableau to visualize key insights.

---

## Dashboard Features

The dashboard includes visualizations such as:

- Sales by product category
- Customer distribution by gender
- Subscription vs non-subscription behavior
- Shipping method preferences
- Purchase trends and comparisons

These visualizations help stakeholders quickly understand business insights.

---

## Key Insights

- Certain product categories generate higher sales.
- Shipping preferences vary across customers.
- Subscription status influences purchasing behavior.
- Customer demographics affect product preferences.

---

## How to Run the Project

1. Download the dataset from this repository.
2. Run the Python scripts/notebooks to perform data cleaning and EDA.
3. Execute SQL queries from the **analysis_queries** folder.
4. Open the **Customer Behavior dashboard (.pbix)** to explore interactive visualizations.
5. Review the presentation slides for summarized insights.
